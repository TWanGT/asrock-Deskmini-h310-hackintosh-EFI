## 使用说明
oc 和 clover 2选一, 将EFI文件整个考入引导分区(包括EFI文件夹)

clover 和 oc 各有优劣
* clover: 引导界面优美, 更加成熟, 配套工具成熟, 配置简单, 文章资料多
* openCore: 目前任然是beta版本, 但是是未来的趋势, 越来越多的补丁作者宣布后续只维护oc版本, 对10.15系统引导时间快不少, 可以更接近原生苹果的体验(包括恢复模式什么的)
